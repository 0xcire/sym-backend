# Sym API

Nest js REST API for sym (sync-your-music)

Taking experience from Propel & internship to hopefully make a decent API.

## Tech

- Node 
- Express
- Nest JS 
- Swagger 

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
